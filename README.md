# Resume

Sanitized public resume site for Jeff Morris, focused on security operations, detection engineering, incident response, analyst enablement, and practical SOC tooling.

This repository is intentionally lightweight. The resume content lives in `resume.md`, and the static page renders it through `index.html` with separate screen and print styles.

## What this shows

* Security operations and incident response experience
* Detection tuning and SIEM workflow experience
* PowerShell, Python, regex, and analyst tooling work
* SOC training, runbooks, and operational documentation
* Sanitized examples of enterprise security work without exposing employer-sensitive details

## Repo structure

```text
.
├── index.html      # Static resume page
├── resume.md       # Resume source content
├── style.css       # Screen styling
└── print.css       # Print/PDF styling
```

## Notes

This is a sanitized public version of my resume. Employer names, internal project details, sensitive workflows, and non-public operational data are intentionally generalized or omitted.

The goal is not to publish internal security work. The goal is to show how I communicate, document, build lightweight tools, and translate SOC work into usable artifacts.
